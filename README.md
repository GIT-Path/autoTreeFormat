# autoTreeFormat
Command line interface that creates easy to read text file of file structure.

# Usage
python autoTree.py -f Example
```
Example
├── build
│   ├── resources
│   │   └── dog.jpg
│   └── src
│       ├── addition.py
│       └── helloworld.py
├── docs
├── License.txt
├── README_FILE_STRUCTURE.txt
├── src
├── tools
├── test
│   ├── benchmarks
│   ├── integration
│   └── unit
└── zz
```
